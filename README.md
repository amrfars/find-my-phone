# 🔍 اعثر على هاتفك - يا عمرو

تطبيق أندرويد بيسمع صوتك في الخلفية، ولما تقول **"يا عمرو"** هيشغل:
- 🔊 صوت عالي (أقصى حجم)
- 🔦 فلاش الكاميرا
- 📳 اهتزاز
- 🔔 صوت تنبيه

## التثبيت على الأندرويد

### المتطلبات
- Linux أو WSL (Windows Subsystem for Linux) أو Mac
- Python 3.8+
- Buildozer

### خطوات البناء

```bash
# ثبّت buildozer
pip install buildozer cython virtualenv

# روح لمجلد المشروع
cd find-my-phone

# ابنِ الـ APK (أول مرة هيحمل SDK و NDK وهيستغرق وقت)
buildozer android debug

# APK هيكون في:
# ./bin/findmyphone-1.0.0-debug.apk
```

### انقل الـ APK للموبايل وثبّته

1. انقل ملف `bin/findmyphone-1.0.0-debug.apk` للموبايل
2. ثبّته (محتاج تسمح بالتثبيت من مصادر خارجية)
3. افتح التطبيق واضغط **"تشغيل الاستماع"**
4. وافق على كل الأذونات (ميكروفون + كاميرا + إشعارات)

## الاستخدام

1. شغّل التطبيق واضغط زر **"تشغيل الاستماع"**
2. التطبيق هيستمع في الخلفية
3. لو هاتفك ضاع، قول **"يا عمرو"** بصوت عالي
4. التطبيق هيشغل الفلاش والصوت والاهتزاز

## ملفات المشروع

| الملف | الوصف |
|-------|-------|
| `main.py` | التطبيق الرئيسي (واجهة Kivy + التحكم) |
| `service.py` | خدمة الخلفية للاستماع المستمر |
| `buildozer.spec` | إعدادات بناء الـ APK |
| `requirements.txt` | مكتبات Python المطلوبة |
| `alarm.mp3` | صوت التنبيه (ضيفه يدوياً) |

## ملاحظات

- **لازم تضيف ملف `alarm.mp3`** في مجلد المشروع عشان صوت التنبيه يشتغل
- الخدمة بتشتغل كـ foreground service عشان الأندرويد ما يقتلش التطبيق
- التطبيق بيستخدم Google Speech Recognition (محتاج إنترنت)
- لو عايز يشتغل بدون إنترنت، ممكن تستخدم Vosk أو Whisper offline
